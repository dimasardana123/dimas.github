Tentang Saya

Halo, saya Mohamad Dimas Ardana Pramudya Putra, seorang pengembang web yang fokus pada pembuatan aplikasi web yang mudah digunakan dan menarik. Saya memiliki minat besar dalam desain UI/UX dan pengembangan frontend. Dengan pengalaman lebih dari 3 tahun, saya telah mengerjakan berbagai proyek, mulai dari website sederhana hingga aplikasi web yang lebih kompleks. Saya selalu berusaha untuk mengikuti perkembangan teknologi terbaru dalam pengembangan web, termasuk teknik desain responsif dan praktik terbaik dalam pengalaman pengguna. Saat ini, saya juga sedang menempuh studi di jurusan Sistem Informasi di Universitas Internasional Semen Indonesia untuk memperdalam pengetahuan saya dalam teknologi dan pengembangan perangkat lunak..
